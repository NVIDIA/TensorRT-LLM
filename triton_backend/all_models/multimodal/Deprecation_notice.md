# ⚠️ DEPRECATION NOTICE ⚠️
Trition backend with multimodal is deprecated and is no longer maintained.

**End of Life:** [TensorRT-LLM-V1.2] \
**Reason:** Trition backend with multimodal uses TensorRT LLM backend which is no longer the default for TensorRT-LLM. \
**Action:** Please use `PytTorch LLM backend` instead. [Link to New Module]
