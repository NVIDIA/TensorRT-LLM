Dynamo K8s Example
=================================

.. toctree::
   :maxdepth: 2
   :caption: Scripts


For Dyanmo K8s example, please follow
[this](https://github.com/ai-dynamo/dynamo/blob/main/docs/guides/dynamo_deploy/quickstart.md)
guide to install Dynamo cloud.  Once the Dynamo cloud is installed, you can use
the [DynamoDeploymentGraph yaml](https://github.com/ai-dynamo/dynamo/tree/main/components/backends/trtllm/deploy)
files to create aggregated, and disaggregated TRT-LLM deployments.
